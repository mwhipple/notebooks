#+title: assignment

Variable assignment is updating the value associated with an
identifier.

This will specifically be used from the perspective of mutable
bindings and a typically side-effect producing statement which changes
such a binding.

This may also be stretched to include assignment style behavior in
languages that do not provide mutable bindings but may provide similar
sugar that provides something like producing nested binding
environments.
