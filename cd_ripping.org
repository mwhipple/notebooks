#+title: cd-ripping

What tools to use nowadays?

This feels like an area that isn't likely to get active attention
other than some integrations to pieces that may still be moving such
as codecs.

* abcde

This is what I was using fairly recently...but the brew formula has since been deprecated :/
https://formulae.brew.sh/formula/abcde
Unfortunately they don't offer an alternative.

At the moment I don't have any deep requirements (FLAC ripping and ideally CD information retrieval),
so I'll see if other options are more readily usable.
