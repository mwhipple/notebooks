#+title: github

GitHub is a widely used [[file:git.org][git]] hosting site with social elements
(probably doesn't need an explanation).

I personally tend to gravitate towards alternatives, but GitHub is
widely used and also what I'm currently using professionally.

GitHub specific functionality can be accessed using the ~gh~ CLI tool.

* Repository Selection

The tool will use the repository for the current project if detected,
therefore a simple setup is to expect to invoke it within a project
working directory.

For notebook use, this will involve having a project-local notebook.
