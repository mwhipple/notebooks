  from deepagents import create_deep_agent

** Accessing Files

A simple option for file access is to make use of the file system backend.
This can then also serve as a context repository.

~virtual_mode~ retains some indirection/control rather than just opening up
the system to the agent.

This is probably not an approach that should typically be used, other
than when starting out in very narrow cases; more flexible and
controlled composition should likely be preferred.

#+begin_src python
  import os
  from deepagents import create_deep_agent
  from deepagents.backends import FilesystemBackend

  boxed_agent = create_deep_agent(
      model="ollama:gpt-oss",
      backend=FilesystemBackend(root_dir=".", virtual_mode=True),
  )
  box_results = boxed_agent.invoke({"messages": [{
      "role": "user",
      "content": "Describe the contents of the root directory, ignoring temp emacs files ending with ~.",
  }]})
  print(box_results["messages"][-1].content)
#+end_src
