#+title: obi

obi is what I'm tentatively calling an effort to expand use of [[id:04B03591-50E7-41E2-A2A5-0C2893FAD39C][org-babel]].
The actual name is just slamming together the ~ob~ used for org-babel
and Star Wars, but it could also theoretically acquire a backronym
like Org Babel Improved.

** Background

I've recently taken to using org babel very heavily...and that mostly
comes down to treating those files as a primary interface/shell. This
therefore seems likely to expand beyond what org-babel may typically
come with org-babel: while all the building blocks seem to be there,
there may be some specific extensions and conventions that
fall out of using it more extensively.

One of the other fairly significant goals is that while I'll be using
obi with org-babel and emacs for the foreseeable future, I'd also want
that to be detachable. Therefore everything should remain fully
transparent so that what is done could be readily extracted to some
other interface like a boring old shell.

Beyond babel and evident alternatives like Jupyter, I'm also likely to
want to draw on ideas from other systems that had interfaces similar
to interactive notebooks such as the Acme editor and Oberon.

** Initial Design Ideas

All execution should default to asynchronous evaluation.
Basic async execution can likely be handled by splitting out the
overall flow, but a further goal would be consistently exposing what
is being done which will be hidden if things are being done directly
in emacs-lisp/system calls, so using something like comint extensively
to allow discovery of the invoked commands will be pursued.

** obi/execute-src-block

This function collects the context which results in an ~info~ object
that is then used to perform the actual execution.

It looks like the key piece to make this a bit more supple is
~org-babel-insert-result~, where if we break this up to support async
flows then that should be trigged by the callback.

#+begin_src emacs-lisp
  (defun obi/execute-src-block (&optional arg info params executor-type)
    "Execute the current source code block and return the result.
  Insert the results of execution into the buffer.  Source code
  execution and the collection and formatting of results can be
  controlled through a variety of header arguments.

  With prefix argument ARG, force re-execution even if an existing
  result cached in the buffer would otherwise have been returned.

  Optionally supply a value for INFO in the form returned by
  `org-babel-get-src-block-info'.

  Optionally supply a value for PARAMS which will be merged with
  the header arguments specified at the front of the source code
  block."
    (interactive)
    (let* ((org-babel-current-src-block-location
            (or org-babel-current-src-block-location
                (nth 5 info)
                (org-babel-where-is-src-block-head)))
  	 (element (org-element-context))
           (info (if info (copy-tree info) (org-babel-get-src-block-info nil element)))
  	 (result-params (cdr (assq :result-params params)))
  	 (exec-start-time (current-time))
  	 (exec-result (obi/exec-result-for-info info)))
      (unless (member "none" result-params)
        (org-babel-insert-result
         exec-result result-params info
         (when (member "replace" result-params) new-hash)
         lang
         (time-subtract (current-time) exec-start-time)))
      (run-hooks 'org-babel-after-execute-hook)
      exec-result))

#+end_src

** obi/exec-result-for-info

~info~ is the object that holds all of the important info (and could be
extended as needed), so let's use that to drive the actual execution.

#+begin_src emacs-lisp
  (defun obi/exec-result-for-info (info)
    (let* ((body (org-babel--expand-body info))
    	 (lang (nth 0 info))
    	 (params (nth 2 info))
    	 (cmd (intern (concat "org-babel-execute:" lang)))
    	 (result-params (cdr (assq :result-params params)))
  	 (maybe-wrapped
  	  (lambda (r)
  	    (let ((wants-value? (eq (cdr (assq :result-type params)) 'value))
  		  (wants-collection? (or (member "vector" result-params) (member "table" result-params))))
  	      (if (and wants-value? wants-collection? (not (listp r)))
  		  (list (list r))
  		r)))))
      (unless (fboundp cmd)
        (error "No org-babel-execute function for %s!" lang))
      (let ((exec-result (funcall maybe-wrapped (save-excursion (funcall cmd body params)))))
        exec-result)))
#+end_src
